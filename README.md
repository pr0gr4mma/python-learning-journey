# python-daily-lab

Daily Python practice + small projects.
Focus: fundamentals (CS50P), practice (freeCodeCamp/Exercism), and practical scripts.

## Structure
- 00_setup: environment notes + quick checks
- 01_basics: fundamentals and small demos
- 02_exercises: course problems and coding practice
- 03_automations: useful scripts (files, logs, web checks, etc.)
- 04_projects: bigger mini-projects

## Setup (Windows/macOS/Linux)
Create and activate a virtual environment:
- Windows:
  python -m venv .venv
  .\.venv\Scripts\activate
- macOS/Linux:
  python3 -m venv .venv
  source .venv/bin/activate

Install dependencies:
  pip install -r requirements.txt

## Run
  python path/to/script.py
